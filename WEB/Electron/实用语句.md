# 实用语句

## 获取 IP

```js
const ip = require("ip");
const IPAddress = ip.address();
console.log(IPAddress);
```

## 获取命令行参数

使用主进程中 process 模块获取
```js
const args = process.argv;
    if (args.length > 0) {
        for (let i of args) {
            i = i.split("=");
            if (i[0] === "url") {
                url = i[1];
            }
        }
    }
```
