# CMD 实用指令

列出所有文件目录
```
dir *.zip  /B >list.txt
```

创建文件与移动文件
```
mkdir "b"
move "Z:\a\a.*" "Z:\b\"
```

获取所有内网 IP
```
arp -a
```

移动文件夹下的所有文件到文件
```
robocopy "E:\a" "E:\b" /move /e
```

把当前目录文件批量重命名
```
@echo off
set a=00
setlocal EnableDelayedExpansion
for %%n in (*.gif) do (
set /A a+=1
ren "%%n" "zy_a_!a!.gif"
)
```

把当前目录文件夹批量重命名，把该bat放在同级目录即可
```
set a=1000
setlocal EnableDelayedExpansion
FOR /f "tokens=*" %%i IN ('dir /a:d /b') DO (
set /A a+=1
ren "%%i" "AA!a!"
)
```

把当前文件夹下的所有压缩文件解压
```
echo off  & setlocal EnableDelayedExpansion
set DIR="%cd%"
echo DIR=%DIR%
for /R %DIR% %%i in (*.rar *.zip *.7z) do ( 
"C:\Program Files\7-Zip\7z.exe" x %%i -o!%%~di%%~pi!
)
pause
```
